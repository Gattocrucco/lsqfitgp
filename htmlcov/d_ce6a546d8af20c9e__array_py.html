116class StructuredArray: 

117 """ 

118 Autograd-friendly imitation of a numpy structured array. 

119  

120 It behaves like a read-only numpy structured array, with the exception that 

121 you can set a whole field/subfield. 

122  

123 Examples 

124 -------- 

125 >>> a = np.empty(3, dtype=[('f', float), ('g', float)]) 

126 >>> a = StructuredArray(a) 

127 >>> a['f'] = np.arange(3) # this is allowed 

128 >>> a[1] = (0.3, 0.4) # this raises an error 

129  

130 Parameters 

131 ---------- 

132 array : numpy array, StructuredArray 

133 A structured array. An array qualifies as structured if 

134 ``array.dtype.names is not None``. 

135  

136 Notes 

137 ----- 

138 The StructuredArray is a readonly view on the input array. When you 

139 change the content of a field of the StructuredArray, however, the 

140 reference to the original array for that field is lost. 

141  

142 """

148 using `d` as `_dict` member (no copy). The shape and size are inferred 

149 from the contents of `d`. 

150 """ 

151 out = super().__new__(cls) 

152 out.dtype = x.dtype 

153 out._dict = d 

154 f0 = x.dtype.names[0] 

155 a0 = d[f0] 

156 subshape = x.dtype.fields[f0][0].shape 

157 out.shape = a0.shape[:len(a0.shape) - len(subshape)]
