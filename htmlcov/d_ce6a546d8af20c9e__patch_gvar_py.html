99  

100 Return 

101 ------ 

102 jac : array 

103 The shape is g.shape + (m,), where m is the total number of primary 

104 gvars that g depends on. 

105 indices : (m,) int array

184  

185 Format a table comparing side by side various collections of gvars. 

186  

187 Parameters 

188 ---------- 

189 *gs : sequence of arrays or dictionaries of gvars 

190 The variables to be tabulated. The structures of arrays and dictionaries 

191 must match. 

192 headers : bool or sequence of strings 

193 If True (default), add automatically an header. If False, don't add an 

194 header. If a sequence with length len(gs) + 1, it contains the column

200 digits.

203 dictionary. Ignored if they are arrays. 

204  

205 Examples 

206 -------- 

207 >>> print(tabulate_together(gvar.gvar(dict(a=1)), gvar.gvar(dict(a=2)))) 

208 key/index value1 value2 

209 --------------------------- 

210 a 1 (0) 2 (0) 

211  

212 See also 

213 -------- 

214 gvar.tabulate 

215  

216 """ 

217 if not gs: 

218 return '' 

219 gs = [g if hasattr(g, 'keys') else numpy.asarray(g) for g in gs] 

220 assert all(hasattr(g, 'keys') for g in gs) or all(not hasattr(g, 'keys') for g in gs) 

221 if keys is not None and hasattr(gs[0], 'keys'): 

222 gs = [{k: g[k] for k in keys} for g in gs] 

223 g0 = gs[0] 

224 if hasattr(g0, 'keys'): 

225 assert all(set(g.keys()) == set(g0.keys()) for g in gs[1:]) 

226 gs = [{k: g[k] for k in g0} for g in gs] 

227 else: 

228 assert all(g.shape == g0.shape for g in gs[1:]) 

229 if g0.shape == (): 

230 gs = [{'--': g} for g in gs] 

231 tables = [ 

232 _splittable(gvar.tabulate(g, headers=['@', ''], ndecimal=ndecimal)) 

233 for g in gs 

234 ] 

235 columns = list(tables[0]) + [t[1] for t in tables[1:]] 

236 if not hasattr(headers, '__len__'):

239 else: 

240 headers = None 

241 else: 

242 assert len(headers) == len(columns) 

243 if headers is not None: 

244 columns = (_head(col, head) for col, head in zip(columns, headers))
