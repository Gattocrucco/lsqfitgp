146 """ 

147  

148 Base class for objects representing covariance kernels. 

149  

150 A Kernel object is callable, the signature is obj(x, y). Kernel objects 

151 can be summed and multiplied between them and with scalars, or raised 

152 to power with a scalar exponent. 

153  

154 Attributes 

155 ---------- 

156 derivable : int or None 

157 How many times the process represented by the kernel is derivable. 

158 ``sys.maxsize`` if it is smooth. ``None`` if the derivability is 

159 unknown. 

160 maxdim : int or None 

161 Maximum input dimensionality. None means unknown. 

162  

163 Parameters 

164 ---------- 

165 kernel : callable 

166 A function with signature ``kernel(x, y)``, where `x` and `y` are 

167 two broadcastable numpy arrays, which computes the covariance of 

168 f(x) with f(y) where f is the Gaussian process. 

169 dim : None or str 

170 When the input arrays are structured arrays, if `dim` is None the 

171 kernel will operate on all fields, i.e., it will be passed the whole 

172 arrays. If `dim` is a string, `kernel` will see only the arrays for 

173 the field named `dim`. If `dim` is a string and the array is not 

174 structured, an exception is raised. If the field for name `dim` has 

175 a nontrivial shape, the array passed to `kernel` is still 

176 structured but has only field `dim`. 

177 loc, scale : scalar 

178 The inputs to `kernel` are transformed as (x - loc) / scale. 

179 forcekron : bool 

180 If True, when calling `kernel`, if `x` and `y` are structured 

181 arrays, i.e., if they represent multidimensional input, `kernel` is 

182 invoked separately for each dimension, and the result is the 

183 product. Default False. The selection indicated by `dim` limits the 

184 dimensions over which `forcekron` applies. 

185 derivable : bool, int, None, or callable 

186 Specifies how many times the kernel can be derived, only for error 

187 checking purposes. True means infinitely many times derivable. If 

188 callable, it is called with the same keyword arguments as `kernel`. 

189 If None (default) it means that the degree of derivability is 

190 unknown. 

191 saveargs : bool 

192 If True, save the all the initialization arguments in a 

193 dictionary under the attribute `initargs`. Default False. 

194 maxdim : int, callable or None 

195 The maximum input dimensionality accepted by the kernel. If 

196 callable, it is called with the same keyword arguments as the 

197 kernel. Default sys.maxsize. 

198 **kw 

199 Additional keyword arguments are passed to `kernel`. 

200  

201 Methods 

202 ------- 

203 rescale : multiply the kernel by some functions 

204 diff : take derivatives of the kernel 

205 xtransf : transform the inputs to the kernel 

206 fourier : take the Fourier series 

207 taylor : take the Taylor series
