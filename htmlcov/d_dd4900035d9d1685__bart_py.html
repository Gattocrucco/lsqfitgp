39 BART kernel. 

40  

41 Parameters 

42 ---------- 

43 alpha, beta : scalar 

44 The parameters of the branching probability. 

45 maxd : int 

46 The maximum depth of the trees. 

47 splits : pair of arrays 

48 The first is an int (p,) array containing the number of splitting 

49 points along each dimension, the second has shape (n, p) and contains 

50 the sorted splitting points in each column, filled with high values 

51 after the length. 

52 gamma : scalar or str 

53 Interpolation coefficient in [0, 1] between a lower and a upper 

54 bound on the infinite maxd limit, or a string 'auto' indicating to 

55 use a formula which depends on alpha, beta, maxd and the number of 

56 covariates, empirically calibrated on maxd from 1 to 3. Default 1

58 pnt : (maxd + 1,) array, optional 

59 Nontermination probabilities at depths 0...maxd. If specified,

61 intercept : bool 

62 The correlation is in [1 - alpha, 1] (or [1 - pnt[0], 1] when using 

63 pnt). If intercept=False, it is rescaled to [0, 1]. Default True. 

64 weights : (p,) array 

65 Unnormalized selection probabilities for the covariate axes. If not 

66 specified, all axes have the same probability to be selected for 

67 splitting. 

68 reset : int or sequence of int, optional 

69 List of depths at which the recursion is reset, in the sense that the 

70 function value at a reset depth is evaluated on the initial inputs for 

71 all recursion paths, instead of the modified input handed down by the 

72 recursion. Default none. 

73  

74 Methods 

75 ------- 

76 splits_from_coord 

77 indices_from_coord 

78 correlation 

79  

80 Notes 

81 ----- 

82 This is the covariance function of the latent mean prior of BART (Bayesian 

83 Additive Regression Trees) [1]_ in the limit of an infinite number of 

84 trees, and with an upper bound :math:`D` on the depth of the trees. This 

85 prior is the distribution of the function 

86  

87 .. math:: 

88 f(\\mathbf x) = \\lim_{m\\to\\infty} 

89 \\sum_{j=1}^m g(\\mathbf x; T_j, M_j), 

90  

91 where each :math:`g(\\mathbf x; T_j, M_j)` is a decision tree evaluated at 

92 :math:`\\mathbf x`, with structure :math:`T_j` and leaf values :math:`M_j`. 

93 The trees are i.i.d., with the following distribution for :math:`T_j`: for 

94 a node at depth :math:`d`, with :math:`d = 0` for the root, the probability 

95 of not being a leaf, conditional on its existence and its ancestors only, is 

96  

97 .. math:: 

98 P_d = \\alpha (1+d)^{-\\beta}, \\quad 

99 \\alpha \\in [0, 1], \\quad \\beta \\ge 0. 

100  

101 For a non-leaf node, conditional on existence and ancestors, the splitting 

102 variable has uniform distribution amongst the variables with any splitting 

103 points not used by ancestors, and the splitting point has uniform 

104 distribution amongst the available ones. The splitting points are fixed, 

105 tipically from the data. 

106  

107 The distribution of leaves :math:`M_j` is i.i.d. Normal with variance 

108 :math:`1/m`, such that :math:`f(x)` has variance 1. In the limit 

109 :math:`m\\to\\infty`, the distribution of :math:`f(x)` becomes a Gaussian 

110 process. 

111  

112 Since the trees are independent, the covariance function can be computed 

113 for a single tree. Consider two coordinates :math:`x` and :math:`y`, with 

114 :math:`x \\le y`. Let :math:`n^-`, :math:`n^0` and :math:`n^+` be the 

115 number of splitting points respectively before :math:`x`, between 

116 :math:`x`, :math:`y` and after :math:`y`. Next, define :math:`\\mathbf 

117 n^-`, :math:`\\mathbf n^0` and :math:`\\mathbf n^+` as the vectors of such 

118 quantities for each dimension, with a total of :math:`p` dimensions, and 

119 :math:`\\mathbf n = \\mathbf n^- + \\mathbf n^0 + \\mathbf n^+`. Then the 

120 covariance function can be written recursively as 

121  

122 .. math:: 

123 \\newcommand{\\nvecs}{\\mathbf n^-, \\mathbf n^0, \\mathbf n^+} 

124 k(\\mathbf x, \\mathbf y) &= k_0(\\nvecs), \\\\ 

125 k_D(\\nvecs) &= 1 - (1 - \\gamma) P_D, 

126 \\quad \\mathbf n^0 \\ne \\mathbf 0, \\\\ 

127 k_d(\\mathbf 0, \\mathbf 0, \\mathbf 0) &= 1, \\\\ 

128 k_d(\\nvecs) &= 1 - P_d \\Bigg(1 - \\frac1{W(\\mathbf n)} 

129 \\sum_{\\substack{i=1 \\\\ n_i\\ne 0}}^p 

130 \\frac{w_i}{n_i} \\Bigg( \\\\ 

131 &\\qquad \\sum_{k=0}^{n^-_i - 1} 

132 k_{d+1}(\\mathbf n^-_{n^-_i=k}, \\mathbf n^0, \\mathbf n^+) 

133 + {} \\\\ 

134 &\\qquad \\sum_{k=0}^{n^+_i - 1} 

135 k_{d+1}(\\mathbf n^-, \\mathbf n^0, \\mathbf n^+_{n^+_i=k}) 

136 \\Bigg) 

137 \\Bigg), \\quad d < D, \\\\ 

138 W(\\mathbf n) &= \\sum_{\\substack{i=1 \\\\ n_i\\ne 0}}^p w_i. 

139  

140 The introduction of a maximum depth :math:`D` is necessary for 

141 computational feasibility. As :math:`D` increases, the result converges to 

142 the one without depth limit. For :math:`D \\le 2` (the default value), the 

143 covariance is implemented in closed form and takes :math:`O(p)` to compute. 

144 For :math:`D > 2`, the computational complexity grows exponentially as 

145 :math:`O(p(\\bar np)^{D-2})`, where :math:`\\bar n` is the average number of splitting 

146 points along a dimension. 

147  

148 In the maximum allowed depth is 1, i.e., either :math:`D = 1` or 

149 :math:`\\beta\\to\\infty`, the kernel assumes the simple form 

150  

151 .. math:: 

152 k(\\mathbf x, \\mathbf y) &= 1 - P_0 \\left( 

153 1 - Q + \\frac Q{W(\\mathbf n)} 

154 \\sum_{\\substack{i=1 \\\\ n_i\\ne 0}}^p w_i 

155 \\frac{n^0_i}{n_i} \\right), \\\\ 

156 Q &= \\begin{cases} 

157 1 - (1 - \\gamma) P_1 & \\mathbf n^0 \\ne \\mathbf 0, \\\\ 

158 1 & \\mathbf n^0 = \\mathbf 0, 

159 \\end{cases} 

160  

161 which is separable along dimensions, i.e., it has no interactions. 

162  

163 References 

164 ---------- 

165 .. [1] Hugh A. Chipman, Edward I. George, Robert E. McCulloch "BART: 

166 Bayesian additive regression trees," The Annals of Applied Statistics, 

167 Ann. Appl. Stat. 4(1), 266-298, (March 2010). 

168 """
