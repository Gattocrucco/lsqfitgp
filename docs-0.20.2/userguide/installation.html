1. Installation¶

First, you have to get a working Python interpreter. There are two main options: the official package from https://www.python.org, and the Spyder IDE. The latter is probably the easier one.

Then, install lsqfitgp by running this command in a shell:

pip install lsqfitgp

1.1. Windows¶

I’ve seen that people may have problems installing the dependency gvar on Windows. This page may help: https://www.lfd.uci.edu/~gohlke/pythonlibs/. Another problematic dependency is JAX, whose Windows support is experimental. I do not test lsqfitgp on Windows.
